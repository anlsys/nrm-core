HBandit.EpsGreedy

Description

This module implements the fixed rate \(\Epsilon\)-Greedy MAB algorithm.

The E(Epsilon)-Greedy algorithm selects a random action with probability \(\Epsilon\), or select the action with best average with probability \(1-\Epsilon\).
