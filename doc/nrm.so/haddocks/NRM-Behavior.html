internal dispatching

nrm :: Time -> NRMEvent -> NRM () #

The nrm function contains the main logic of the NRM daemon. It changes the state and produces an associated behavior to be executed by the runtime. This contains the slice management logic, the sensor callback logic, the control loop callback logic. Works in the @NRM monad.
